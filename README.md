A bulletproof BackgroundService implementation with auto-resurrection and heartbeat monitoring.

## Overview

Lazarus provides a resilient `BackgroundService` derived implementation designed to be bulletproof. Features include:

- **Auto-resurrection**: Automatically recovers from failures
- **Heartbeat monitoring**: Ensures services remain healthy
- **Configurable retry policies**: Customise resurrection behaviour

## Installation
